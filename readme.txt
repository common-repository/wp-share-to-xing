== Description ==

This plugin will install a XING-Button, to share a link to your XING-Contacts, to your blog articles, pages, archives, categories and rss.

**Features**

* Easy install
* Panel for easy configuration
* Config display locations (entries, pages, home page, archive, category)
* Config position in content (before, both, shortcode)

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'WP-Share to XING'
1. Click 'install'

**Manual Installation**

1. Upload folder `wp-share-to-xing` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure plugin in 'Share to XING' menu
